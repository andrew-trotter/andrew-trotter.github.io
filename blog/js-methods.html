What is the .map() method

The .map() method allows us to call a function on every item in the given array. It will output as a new array and doesn't alter the original array. It functions like a for loop that outputs a new array. Here's an example of how it functions under the hood:

// Array of students:
let students = ['Andrew', 'Adam', 'Jane', 'Sarah', 'Valentine']
// Custom Map function:
function customMap(arr, func) {
  let newArr = [] // empty array to push new stuff into
  for (let i = 0; i < arr.length; i++) {
    let item = arr[i]
    newArr.push(func(item)) // calling the "func" function on the array item
  }
  return newArr // returning new "mapped" array
}
// calling our customMap function:
let mappedArr = customMap(students, (name) => {
  return {
    name: name,
  }
})

console.log(mappedArr)
  // Should return: {
    name: 'Andrew',
    name: 'Adam',
    name: 'Jane',
    name: 'Sarah',
    name: 'Valentine',}

The customMap function has two parameters. "arr" which is the array to work with, and "func" which is the operation to perform on the array.
It makes a new array and in a for loop, makes an item and pushes the result of the item from the function to the new array. Then, it returns a new array with the results of the function. In this case we call it to make a student object.

What is the .join() method?

This method allows you to concatenate all elements in an array and output as a string. You're able to use a parameter to put in between each element when output as a string.

In this example, we have an array of users with names stored as separate elements that I would like to make user names for. The usernames will have a hyphen (-) between the first and last name.

Example: generateUserNames([[`Margaret`, `Phillips`], [`Andy`, `Johnson`], [`Sarah`, `Conner`]])

let people = [
  [`Margaret`, `Phillips`],
  [`Andy`, `Johnson`],
  [`Sarah`, `Conner`],
]

function generateUserNames(array) {
  let userNames = array.map((person) => person.join(`-`))
  return userNames
}

console.log(generateUserNames(people))
// Should return: ['Margaret-Phillips', 'Andy-Johnson', 'Sarah-Conner']

What is Number()?

This is not a method, but a constructor to work with values. It is used to coerce or turn other data types into a number. Usually it is used to turn string into numbers, but can also turn Boolean values into "0" or "1". If it fails it returns (NaN)

In this example, we have an array full of numbers that are boxes of chocolates, but they were stored as string values. I want to know how many chocolates were sold, so I need the values as numbers.

let stringNumbers = ['1', '5', '11', '55']

function convertStringToNumArray(stringArray) {
  let numArray = stringArray.map((num) => Number(num))
  let sum = sumArray(numArray)
  return sum
}
function sumArray(arr) {
  return arr.reduce((a, b) => a+b);
}

console.log(convertStringToNumArray(stringNumbers))
//Expected output: 72
//Output without Number(): "151155"
